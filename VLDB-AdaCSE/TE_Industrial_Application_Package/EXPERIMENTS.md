# TE Experiments

This file documents the executable parts of the TE industrial application
artifact.

## 1. Environment

- MATLAB R2020a or newer is recommended.
- Add the full `VLDB-AdaCSE` directory to the MATLAB path.
- The Bayes Net Toolbox (BNT) included in this repository is used directly.

Initialization:

```matlab
cd('VLDB-AdaCSE');
addpath(genpath(pwd));
```

## 2. Input Data

- Raw TE files:
  - `Data/raw/datasetTE.csv`
  - `Data/raw/TEGroundTruth.txt`
- Prepared TE input:
  - `Data/prepared/te_real_discrete.mat`

The prepared discrete TE file is generated by:

```matlab
prepare_real_te_data();
```

## 3. Main Comparison Experiment

Run the TE multi-method comparison used in the paper:

```matlab
[all_results, summary_file] = run_real_te_comparison_experiment(10, 100, 200, 7, ...
    {'AdaCSE','PSX','EKGA-std','AESL-GA','hybrid-SLA','MIGA','MAGA'});
```

This command produces:

- `Results/TE_Real_Application_Batch/real_te_structure_metrics_all_trials.csv`
- `Results/TE_Real_Application_Batch/real_te_structure_metrics_mean_std.csv`
- learned DAG files under `Results/TE_Real_Application_Batch/learned_dags/`

## 4. Exact-Direction Recalculation

For direct comparison with IGES-RCI:

```matlab
recompute_te_metrics_tkde(fullfile(pwd, 'TE_Industrial_Application_Package', ...
    'Results', 'TE_Real_Application_Batch'));
```

This command produces:

- `Results/TE_Real_Application_Batch/real_te_structure_metrics_tkde_all_trials.csv`
- `Results/TE_Real_Application_Batch/real_te_structure_metrics_tkde_mean_std.csv`

## 5. Exploratory Downstream Analysis

The saved learned DAGs can also be reused for downstream-ranking exploration:

```matlab
[all_rows, summary_file] = run_real_te_application_batch_analysis( ...
    fullfile(pwd, 'TE_Industrial_Application_Package', 'Results', ...
    'TE_Real_Application_Batch'), {'X1','X22'}, 3);
```

These outputs are preserved in `Results/TE_Real_Downstream_Ranking/`, but are
not part of the final paper's main claims.
